# HTML Authentication POC

This project is a simple HTML authentication system created as part of an assignment.

## Pages included

- login.html
- register.html
- forgot-password.html
- reset-password.html
- dashboard.html

## Features

- User registration form
- User login form
- Forgot password functionality
- Reset password page
- Dashboard page
- Proper navigation using anchor tags
- Form validation using HTML
